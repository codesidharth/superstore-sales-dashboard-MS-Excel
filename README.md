# Superstore Sales Dashboard

## 📊 Project Overview
This project is an interactive sales dashboard created using Microsoft Excel. It analyzes sales performance across categories, regions, and customers.

## 🚀 Features
- Pivot Tables for data analysis
- Interactive charts (bar, line)
- Slicers for dynamic filtering
- KPIs (Total Sales, Total Orders, Average Sales)

## 🔍 Key Insights
- Technology category has highest sales
- West region generates maximum revenue
- Top customers contribute major sales

## 🛠 Tools Used
- Microsoft Excel
